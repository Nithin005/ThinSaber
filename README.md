# ThinSaber

A Mod for customizing Width of the Saber. Tested with BeatSaber Version 1.31.1.

# Dependancies

* BS Utils

* BSML

# How to install?

Download `ThinSaber.dll` from releases, paste it in the `Plugins` folder of BeatSaber Game Directory.
